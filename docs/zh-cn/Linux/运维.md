
## 查看进程
```bash 
netstat -lntp    #查看监听(Listen)的端口

netstat -antp   #查看所有建立的TCP连接

netstat -tulpn  #查看所有运行中的服务的详细信息

ps -ef              #显示所有进程

ps -aux           #显示使用内存的进程

top                  #查看内存使用说明 (shift+m 按照排名)
```

## 替换文件换行符
windows编辑过文件可导致脚本启动问题,通过此命令替换
```bash
sed -i 's/\r$//' check_tool.sh
```
